== Description ==

Install on your MainWP dashboard site and allows you to add additional options for your storing information against client sites
== Installation ==

1. Upload the `mainwp-field-manager` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings it is located in the top level menu MainWP Field Manager
